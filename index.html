What is DOM ?

DOM (Document object model) is a programming interface for SVG documents, XML, HTML . It is a standard which is followed by Browser, javascript engines, XML processors etc.

In general DOM is tree structure composed of grouped nodes. It provides an approch way of retrieving, storing and modifying these nodes.

The framworks on which by default HTML works is XML Extensible markup Language, DOM -Example useing XML File.
